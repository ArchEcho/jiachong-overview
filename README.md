## 安装方法

### 方法1（推荐）

加入游戏内频道“大天使边境要塞”，一键导入。

### 方法2

下载yaml文件到我的文档/EVE/overview，进入游戏总览设定>Misc，先Reset All Overview Settings，然后Import Overview Settings。
在导入窗口中选择Check All。
